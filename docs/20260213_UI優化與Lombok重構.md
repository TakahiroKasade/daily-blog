# 2026/02/13 - Phase 5 UI 優化與功能重構

## 1. 主要變更 (Feature Changes)
### 1.1 Lombok 重構 (Refactoring)
- **Entities**: 為 `Post`, `Experience`, `User`, `Project` 引入 Lombok。
- **Annotations**: 使用 `@Data`, `@NoArgsConstructor`, `@AllArgsConstructor` 取代傳統的 Getter/Setter/Constructor。
- **效益**: 減少約 100 行重複程式碼，提升可讀性。

### 1.2 RWD 手機版適配 (Responsive Design)
- **Navbar**: 
  - 將導覽列重構為 Bootstrap 5 標準 `navbar-collapse` 結構。
  - 新增漢堡選單 (Hamburger Menu)，解決手機版按鈕破版問題。
- **CSS**: 
  - 新增 `responsive.css`，針對手機 (`max-width: 768px`) 調整 `hero-title` 字體大小與 `avatar` 尺寸。
  - 優化 Resume Timeline 在小螢幕上的 Padding 與間距。

### 1.3 文章分類功能 (Category Feature)
- **Backend**:
  - `Post.java`: 新增 `category` 欄位 (String)，預設值為 "Tech"。
  - `DataSeeder.java`: 更新測試資料生成邏輯，加入預設分類。
- **Frontend**:
  - `create_post.html` & `edit_post.html`: 新增分類下拉選單 (`<select>`)。
  - `index.html`: 在文章卡片標題旁顯示分類標籤 (`.badge`)。
