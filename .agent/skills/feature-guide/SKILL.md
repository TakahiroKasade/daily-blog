# Feature Guide Skill (新手導覽功能清單)

## 概述
以**資深工程師的視角**，為菜鳥新手工程師分析整個專案，產出結構化的功能清單。
- 依照難度由低到高排列
- 標示前端、後端、資料庫的串聯關係
- 幫助新手快速掌握專案脈絡

## 使用方式

### 生成完整功能導覽
```
請幫我分析專案功能，產出新手導覽清單
```

### 指定模組分析
```
請幫我分析 [模組名稱] 的功能脈絡
```

### 快速上手指南
```
請幫新人整理這個專案的學習路徑
```

## 分析步驟

當執行此 skill 時，我會：

1. **掃描專案結構**
   - 列出所有 Controller、Service、Repository
   - 識別前端頁面與 JS 檔案
   - 分析資料庫 Entity

2. **識別功能模組**
   - 根據 Controller 端點識別功能
   - 追蹤 API 到前端的調用關係
   - 識別資料庫表的關聯

3. **評估難度等級**
   - ⭐ 入門: 靜態頁面、簡單展示
   - ⭐⭐ 基礎: 單表 CRUD
   - ⭐⭐⭐ 中階: 多表關聯、業務邏輯
   - ⭐⭐⭐⭐ 進階: 認證授權、複雜查詢
   - ⭐⭐⭐⭐⭐ 專家: 性能優化、架構設計

4. **建立串聯關係圖**
   - 前端 → API → Controller → Service → Repository → DB

5. **產出導覽文件**
   - 儲存至 `docs/FEATURE_GUIDE.md`
